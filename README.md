# Task 2: Exploratory Data Analysis (EDA) - Business Intelligence

## 📌 Project Overview

This project demonstrates **Exploratory Data Analysis (EDA)** techniques to understand, analyze, and visualize a business dataset. The objective is to uncover meaningful insights, identify patterns, detect anomalies, and support data-driven business decisions using Python.

---

## 🎯 Objectives

* Understand the structure of the dataset.
* Clean and preprocess the data.
* Perform exploratory data analysis.
* Visualize important trends and relationships.
* Generate business insights from the analysis.

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/anithapappala20-ui/Task-2-Exploratory-Data-Analysis-EDA-Business-Intelligence.git
```

### 2. Navigate to the project folder

```bash
cd Task-2-Exploratory-Data-Analysis-EDA-Business-Intelligence
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the notebook and run all cells.
